# 📊 Telco Customer Churn Prediction using Deep Learning (ANN)

A Deep Learning project that predicts whether a telecom customer is likely to churn based on customer demographics, subscription details, and billing information.

Built using **TensorFlow, Keras, Streamlit, Pandas, NumPy, and Scikit-learn**.

---

## 🚀 Live Features

- Customer Churn Prediction
- Artificial Neural Network (ANN)
- Streamlit Web Application
- Data Preprocessing
- Label Encoding
- One-Hot Encoding
- Feature Scaling
- Probability Prediction
- Modern User Interface

# 📦 Installation

Clone the repository

```bash
git clone https://github.com/aayushpatidar0924-dotcom/telco-customer-churn-prediction-ann
```

Move into project

```bash
cd Telco-Customer-Churn
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
